🛡️ Simple Password Manager

Консольный менеджер паролей на Python. Позволяет быстро генерировать сложные пароли, сохранять их в локальный файл и управлять списком.
🚀 Основные возможности

    Генерация паролей: Использует комбинацию букв (регистр учитывается), цифр и спецсимволов.

    Хранение: Автоматическое сохранение в файл passwords.txt.

    Управление: Просмотр всех записей с нумерацией и возможность удаления по индексу.

🛠 Разбор кода
Генерация пароля

Для создания случайной последовательности используется стандартная библиотека random и набор символов string:
Python

def generate_password(length):
    chars = string.ascii_letters + string.digits + string.punctuation
    return "".join(random.choice(chars) for _ in range(length))

Работа с файлами

Скрипт использует контекстные менеджеры with open(), что гарантирует правильное закрытие файла даже при возникновении ошибок:

    mode="a" — добавление нового пароля в конец файла.

    mode="r" — чтение списка.

    mode="w" — перезапись файла при удалении записи.

📋 Инструкция по использованию
Запуск

Убедитесь, что у вас установлен Python 3.6 или выше:
Bash

python3 smt.py

Интерфейс

После запуска откроется текстовое меню:

    Generate password — введите желаемую длину (например, 16), и пароль будет создан и сохранен.

    Show passwords — выводит пронумерованный список всех ваших паролей.

    Delete password — введите номер строки из списка, чтобы удалить её.
